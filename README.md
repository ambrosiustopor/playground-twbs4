# playground-twbs4

Playground for Twitter Bootstrap 4 components, using Vue.js.

![Screenshot of playground](screenshot.png)


## Getting Started

Install dependencies:

```sh
bower install
```

Open `htdocs/index.html` in web browser.

To compile (style files) from source:

```sh
gulp
```
